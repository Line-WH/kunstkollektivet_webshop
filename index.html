Vores værdier

Fællesskab

KunstKollektivet er bygget op omkring mennesker. Kunder, kunstnere og samarbejdspartnere er ikke bare en del af virksomheden. Uden dem ville vi ikke findes.

Ansvarlig produktion

Print kræver resourcer og det tager vi ansvar for. Derfor er vi transparente i hvordan vi laver vores produkter og vi arbejder løbende på at bruge mere ansvarlige materialer, emballage og processer.

Transparens

Vi sælger ikke AI-genererede produkter. Vores værker er skabt af mennesker og vi ønsker at gøre processen bag mere tydelig, så både kunder og kunstnere kan føle sig mere trygge.
